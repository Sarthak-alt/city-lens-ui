# 🏙 City Lens — Smart City Complaint Platform (UI Prototype)

## 📌 Overview
This is a UI/UX prototype of **City Lens**, an AI-powered smart city complaint platform.

It demonstrates how citizens can report issues using images, and how the system visualizes, tracks, and manages complaints.

---

## 🎯 Purpose
- Showcase product design & UI skills
- Demonstrate system flow for a smart city platform
- Prototype before full-stack development

---

## ✨ Features (UI Simulation)
- 📸 Photo upload interface with AI analysis preview
- 🧮 Priority scoring visualization
- 🗺 Map-based complaint tracking UI
- 📊 Admin dashboard layout
- 📋 Complaint tracking system
- 🔐 Authentication screens

---

## 🖥 Tech Used
- HTML
- CSS (Modern UI + responsive design)
- JavaScript (UI interactions)
